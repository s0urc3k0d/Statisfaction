# Référence des variables d’environnement

## Web (`apps/web/.env.local`)
- NEXT_PUBLIC_API_BASE_URL — URL API (ex: https://api.statisfaction.sourcekod.fr)
- NEXT_PUBLIC_SITE_URL — URL publique du site (ex: https://statisfaction.sourcekod.fr)
- NEXT_PUBLIC_ADSENSE_CLIENT — ca-pub-*
- NEXT_PUBLIC_ADSENSE_SLOT_HOME_HERO, NEXT_PUBLIC_ADSENSE_ENABLE_HOME_BEFORE_FOOTER, NEXT_PUBLIC_ADSENSE_SLOT_HOME_BEFORE_FOOTER
- NEXT_PUBLIC_ADSENSE_SLOT_ABOUT_MID, NEXT_PUBLIC_ADSENSE_SLOT_ABOUT_FOOT
- NEXT_PUBLIC_ADSENSE_SLOT_FEATURES_HERO, NEXT_PUBLIC_ADSENSE_SLOT_FEATURES_MID

## Server (`apps/server/.env`)
- PORT, SESSION_SECRET, BASE_URL, FRONTEND_URL
- PUBLIC_SITE_URL — URL publique du site (liens absolus/emails)
- Redis (optionnel): REDIS_URL
- Twitch: TWITCH_CLIENT_ID, TWITCH_CLIENT_SECRET, TWITCH_REDIRECT_URI
- EventSub: EVENTSUB_SECRET, EVENTSUB_CALLBACK
- SMTP: SMTP_HOST, SMTP_PORT, SMTP_SECURE, SMTP_USER, SMTP_PASS, SMTP_FROM
